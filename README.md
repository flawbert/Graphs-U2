Nesse repositório foram implementados algoritmos de busca pela Árvore Geradora Mínima (Kruskal, Prim, Chu-Liu/Edmonds), pelo Caminho Mais Curto (Dijkstra, Bellman-Ford,Floyd-Warshall) e pelos Grafos Eulerianos (Hierholzer ciclos e caminhos).
